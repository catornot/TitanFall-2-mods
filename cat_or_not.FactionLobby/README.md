# LobbyFaction

this mod puts **your faction leader** next to you

to change the rotation speed or faction

go to `...mod/cfg/autoexec_faction_lobby.cfg`

If you don't see any changes try to reloading your lobby or type `map mp_lobby` in the console

now comes with some weapons and spinning on all axis

now 100% client sided :)

**Enjoy!**
